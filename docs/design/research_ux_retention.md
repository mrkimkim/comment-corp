## 1. 리텐션 베스트 프랙티스 (2025-2026)

### 1.1 업계 벤치마크

| 지표 | 평균 (캐주얼) | 상위 25% | 우리 목표 |
|------|-------------|---------|----------|
| D1 리텐션 | 26~28% | 45%+ | 40%+ |
| D7 리텐션 | ~8% | 20%+ | 18%+ |
| D30 리텐션 | ~3% | 10%+ | 8%+ |

> UA(유저 획득) 비용이 급등하면서 D1 50% 이상이 새로운 기준으로 부상. D30은 장기 수익의 핵심 예측 지표로, 진정한 습관 형성(habit formation)과 게임-시장 적합성(game-market fit)을 반영.

### 1.2 D1 리텐션 (첫날 이탈 방지)

**핵심 원칙: 60초 안에 코어 게임플레이 진입**

- 튜토리얼은 최소한으로, 인터랙티브하게. 설정 화면/계정 생성을 앞세우지 않음
- 첫 번째 게임 세션에서 "아하! 모먼트"를 제공 (첫 악플 차단 성공의 쾌감)
- 온보딩 중 작은 보상 연속 제공 (첫 콤보, 첫 완주, 첫 등급 획득)

**Comment Corp 적용:**
- 첫 실행 시 5개 댓글로 구성된 "미니 튜토리얼 라운드"(30초) 진행
- 좌/우 스와이프 한 번씩만 가르치고, 나머지는 플레이 중 자연스러운 팝업으로 학습
- 첫 게임 클리어 시 반드시 C등급 이상 보장 (성취감)

### 1.3 D7 리텐션 (중기 이탈 방지)

**핵심 원칙: 중급 콘텐츠 & 소셜 기능 노출**

- D2-D7 구간에서 새로운 셀럽 타입, 아이템, 이벤트를 단계적 해금
- PvP, 소셜 기능, 이벤트를 이 시점에 도입하여 "다른 사람과 함께하는 경험"으로 전환
- 2~4주 주기의 콘텐츠 업데이트가 후반 이탈률을 유의미하게 감소시킴

**Comment Corp 적용:**
- D1: 아이돌 타입만 해금 -> D2: 배우 -> D3: 유튜버 -> D5: 스포츠 -> D7: 정치인
- D3 시점에 리더보드 기능 해금 및 안내
- D5 시점에 첫 이벤트 참여 유도

### 1.4 D30 리텐션 (장기 습관 형성)

**핵심 원칙: 진행감(progression), 콘텐츠 깊이, LiveOps**

- 코어 루프, 진행 시스템, 콘텐츠 깊이가 지속적 참여를 유지하는지가 관건
- 플레이타임이 리텐션과 LTV의 가장 신뢰할 수 있는 선행 지표
- 시간/행동 기반 퀘스트("오늘 30분 플레이하기", "적 100명 처치하기")가 명확한 목표 제공

**Comment Corp 적용:**
- 시즌 시스템 도입 (4주 단위)
- 장기 업적 체계 (누적 1,000개 악플 차단, 전 타입 S등급 등)
- 주간 챌린지 + 시즌 보상으로 반복 방문 유도

### 1.5 A/B 테스팅

온보딩 플로우, 보상 시스템, 난이도 곡선, UI/UX 요소의 변형을 체계적으로 비교 테스트하여 리텐션 최적화. Firebase Remote Config + A/B Testing 활용 권장.

## 2. 수익화 모델 분석

### 2.1 모델별 비교

| 모델 | 장점 | 단점 | 댓글 주식회사 적합도 |
|------|------|------|-------------------|
| **프리미엄(유료 앱)** | 깔끔한 수익, 광고 없음 | 다운로드 장벽, 캐주얼 시장 부적합 | ★★☆☆☆ |
| **F2P + 광고 전용** | 진입 장벽 0, 높은 DAU | 유저 피로, 수익 천장 낮음 | ★★★☆☆ |
| **F2P + IAP 전용** | 고과금 유저 수익 큼 | 구매율 3.5%, P2W 리스크 | ★★☆☆☆ |
| **F2P + 하이브리드(광고+IAP)** | 두 수익원 시너지, 2026 트렌드 | 설계 복잡성 | ★★★★★ |
| **배틀패스/시즌패스** | 전환율 12~18%, 정기 수익 | 콘텐츠 제작 부담 | ★★★★☆ |

### 2.2 권장 모델: F2P 하이브리드 (광고 + 소액 IAP + 시즌패스)

**근거:**
- F2P 게임이 모바일 게임 수익의 97%를 차지
- 하이브리드 수익화가 2026년 가장 효과적인 전략으로 부상
- 캐주얼/퍼즐 게임은 광고:IAP = 50:50 하이브리드가 주류
- 보상형 광고(Rewarded Video)가 광고 수익의 62%를 차지하며, 유저 참여율 45-60%

**구체적 설계:**

| 수익원 | 내용 | 예상 비중 |
|--------|------|----------|
| 보상형 광고 | 게임 오버 후 "부활" 광고, 결과화면에서 "2배 보상" 광고 | 50% |
| 배너/전면 광고 | 결과 화면 하단 배너, 3게임마다 전면 광고 (초반에는 빈도 낮게) | 15% |
| 시즌패스 | 월 $2.99~$4.99, 시즌 전용 스킨/뱃지/보너스 보상 | 25% |
| 소액 IAP | 광고 제거 $3.99 (일회성), 스킨팩 $0.99~$1.99 | 10% |

**핵심 원칙:**
- 초반(D1-D3)에는 광고 빈도 최소화 -> 게임에 빠진 후 점진적 증가
- 절대 P2W(Pay-to-Win)하지 않도록 설계: 아이템은 게임 내 획득 충분, 구매는 "시간 절약" 또는 "꾸미기"
- 가장 성공적인 게임은 지출을 강요하지 않고, 좋은 설계와 LiveOps로 초대(invite)함

## 3. 게임 심리학 활용

### 3.1 보상 루프 (Reward Loop)

**원리:** 도파민은 보상을 "받을 때"보다 "기대할 때" 더 많이 분비됨. 게임은 기대-행동-보상의 루프를 설계하여 반복 플레이를 유도.

**Comment Corp 적용:**
- **즉각적 피드백**: 스와이프 정답 시 0.3초 내 시각/청각 피드백 (체크마크 Lottie + "띵" 효과음)
- **콤보 에스컬레이션**: 콤보 5/10/20/30에서 점점 강렬한 시각 이펙트 (불꽃 -> 폭발 -> 무지개)
- **피버 진입 보상**: 피버 모드 진입 시 화면 전체 이펙트 + 환호 사운드 = 도파민 폭발 순간

### 3.2 변동 보상 (Variable Rewards)

**원리:** 고정 보상보다 랜덤/반랜덤 보상이 훨씬 강력한 동기 부여. 예측 불가능한 보상 시스템이 도파민 반응을 극대화.

**Comment Corp 적용:**
- 게임 종료 후 "럭키 뽑기": 결과 화면에서 3장의 카드 중 1장 선택 -> 코인/스킨 조각/보너스 XP 중 랜덤
- 콤보 30 이상 달성 시 "레어 보상 드롭": 한정판 뱃지나 프로필 프레임 랜덤 획득
- 일일 첫 게임 보너스: 매일 다른 보상 (예측 불가)

### 3.3 진행감 (Progression)

**원리:** 경험치, 레벨업, 잠금 해제 등이 "발전하고 있다"는 감각을 제공. 명확한 목표가 집중력을 유지시킴.

**Comment Corp 적용:**
- **플레이어 레벨 시스템**: 게임 XP 누적 -> 레벨업 -> 새 컨텐츠 해금
- **셀럽 타입별 숙련도**: 각 타입 플레이 횟수에 따라 브론즈->실버->골드->다이아->마스터 등급
- **진행 바 시각화**: 메인 화면에 "다음 해금까지 OO XP" 진행 바 항상 표시

### 3.4 손실 회피 (Loss Aversion)

**원리:** 사람은 같은 크기의 이득보다 손실을 약 2배 더 강하게 느낌. 스트릭(연속) 메커니즘은 "끊기기 싫다"는 심리를 활용.

**Comment Corp 적용:**
- **데일리 로그인 스트릭**: 7일 연속 로그인 시 특별 보상. 단, "스트릭 보호권" 제공 (1일 미접속 허용)
- **멘탈 게이지 긴장감**: 멘탈이 30% 이하로 떨어지면 화면 붉어짐 + 심장박동 SFX = 잃고 싶지 않은 감정 극대화
- **하이스코어 위협**: "당신의 기록을 OO님이 넘었습니다" 푸시 알림

### 3.5 윤리적 설계 원칙

보상 루프와 FOMO를 활용하되, 다음을 준수:
- 스트릭 리셋 시 즉시 "다시 시작하기 쉬운" 첫 보상 제공 (절망감 방지)
- 광고 시청을 강제하지 않음 (항상 "건너뛰기" 제공)
- 과금 없이도 모든 게임플레이 콘텐츠 접근 가능 (꾸미기만 유료)
- 과도한 플레이 경고 (2시간 연속 플레이 시 부드러운 알림)

## 4. 소셜 기능이 리텐션에 미치는 영향

### 4.1 핵심 통계

- 상위 모바일 게임의 2/3가 최소 1개 이상의 소셜 기능을 포함
- 활발한 소셜 참여를 하는 유저는 그렇지 않은 유저보다 플레이타임이 유의미하게 높음
- 소셜 연결이 있는 플레이어는 높은 리텐션율을 보임

### 4.2 기능별 영향 분석

| 소셜 기능 | 리텐션 효과 | 구현 비용 | 댓글 주식회사 적합도 |
|-----------|-----------|----------|-------------------|
| **글로벌 리더보드** | 중간 (경쟁심 자극) | 낮음 (이미 구현) | ★★★★★ |
| **세분화 리더보드** | 높음 (국가/친구/주간) | 중간 | ★★★★★ |
| **친구 대결** | 매우 높음 (직접적 경쟁) | 높음 | ★★★★☆ |
| **결과 공유** | 중간 (바이럴 + 자랑) | 낮음 | ★★★★★ |
| **커뮤니티 이벤트** | 높음 (한정 참여 = FOMO) | 중간 | ★★★★☆ |
| **길드/클랜** | 매우 높음 (소속감) | 매우 높음 | ★★☆☆☆ |

### 4.3 Comment Corp 소셜 전략 (우선순위)

**Phase 1 (출시 포함):**
1. 세분화 리더보드: 글로벌 / 국가 / 주간 (리셋으로 경쟁 활성화)
2. 결과 공유: "나는 S등급 셀럽 수호자! 콤보 47x" + 게임 스크린샷을 SNS 공유 (카카오톡, Instagram, X)
3. 공유 시 "초대 코드" 삽입 -> 초대받은 친구 가입 시 양쪽 보상

**Phase 2 (출시 후 1개월):**
4. 친구 리더보드: Game Center / Google Play Games 친구 연동
5. "도전장 보내기": 특정 셀럽 타입+난이도로 친구에게 점수 도전

**Phase 3 (출시 후 3개월):**
6. 주간 토너먼트: 매주 특정 조건(예: 정치인 타입만, 아이템 없이)으로 전체 유저 경쟁
7. 시즌 종합 랭킹

## 5. 구체적 아이디어 목록

### 5.1 데일리 챌린지 시스템

> 매일 고유한 조건의 특별 게임 모드를 제공하여 일일 복귀 동기 부여

**상세 설계:**
- 매일 00:00(UTC) 갱신되는 1개의 특별 챌린지
- 조건 예시: "아이템 없이 90초 생존", "유튜버 타입에서 콤보 20 달성", "정확도 95% 이상 클리어"
- 클리어 시 일일 코인 + 스트릭 보상
- 7일 연속 클리어 시 "주간 보너스 상자" (랜덤 보상)
- 미완료 챌린지는 넘어가되, 스트릭만 리셋 (스트릭 보호권으로 1회 방어 가능)

| 태그 | 값 |
|------|-----|
| 구현 난이도 | 3/5 |
| 리텐션 기대치 | 5/5 |
| 심플함 유지 | Y |

### 5.2 시즌 시스템 (시즌패스 포함)

> 4주 단위 시즌으로 목표와 보상을 리프레시하여 장기 플레이 유도

**상세 설계:**
- 시즌 기간: 4주 (연 13시즌)
- 시즌 테마: "아이돌 컴백 시즌", "월드컵 시즌", "총선 시즌" 등 시의성 있는 테마
- 무료 트랙: 시즌 XP 적립 -> 30단계 보상 (코인, 기본 스킨, 뱃지)
- 유료 트랙 ($2.99~$4.99): 프리미엄 보상 추가 (한정 스킨, 프로필 프레임, 이모티콘)
- 시즌 종료 시 리더보드 리셋 + 시즌 최종 등급 뱃지 영구 지급
- "시즌 미션": 주간 미션 6개 + 시즌 전체 미션 3개

| 태그 | 값 |
|------|-----|
| 구현 난이도 | 4/5 |
| 리텐션 기대치 | 5/5 |
| 심플함 유지 | Y (UI는 간결하게, 보상 트랙 1개 화면) |

### 5.3 업적/뱃지 시스템

> 다양한 플레이 목표를 제시하여 탐험적 플레이와 장기 목표 제공

**상세 설계:**
- 카테고리별 업적:
  - **수호자**: 악플 차단 누적 (100/500/1000/5000/10000)
  - **정확도**: 정확도 90%/95%/99%/100% 달성
  - **콤보 마스터**: 콤보 10/20/30/40/50 달성
  - **서바이벌**: 120초 완주, 멘탈 50% 이상 완주, 무피격 완주
  - **타입 전문가**: 각 셀럽 타입에서 S등급 달성
  - **아이템 장인**: 각 아이템 100회 사용
  - **피버 킹**: 피버 모드 누적 50/100/200회 진입
  - **히든**: 특정 조건 발견 시 해금 (예: 첫 댓글에서 즉시 틀리기)
- 각 업적에 고유 뱃지 아이콘 + XP 보상
- 뱃지를 프로필에 최대 3개 장착하여 리더보드에 표시
- 업적 달성 시 공유 가능한 카드 이미지 자동 생성

| 태그 | 값 |
|------|-----|
| 구현 난이도 | 2/5 |
| 리텐션 기대치 | 4/5 |
| 심플함 유지 | Y |

### 5.4 커스터마이징 시스템

**상세 설계:**
