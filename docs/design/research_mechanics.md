## 1. 스와이프 기반 게임 분석

### 1-1. Reigns (2016~, Nerial)

**핵심 메카닉:**
- Tinder 스타일 좌/우 스와이프로 왕국 통치 결정
- 4개 자원(교회/국민/군대/재정) 밸런스 유지 -- 하나라도 0 또는 MAX에 도달하면 게임 오버
- 약 887장의 고유 카드, 다단계 이벤트 체인

**중독성의 원인:**
- **"한 판만 더" 루프**: 죽음이 곧 새로운 왕의 즉위 -> 실패가 벌이 아니라 콘텐츠 해금 수단
- **숨겨진 정보 디자인**: 카드가 변화의 *크기*는 보여주지만 *방향*은 숨김 -> 매 스와이프가 도박적 쾌감
- **단순함 속 깊이**: 원터치 조작이지만, 이전 선택에 따라 카드 풀이 동적으로 바뀌어 "모든 카드가 의미 있어 보이는" 착각 유발
- **속도/무게감 대비**: 스와이프는 0.3초, 결과는 왕국 전체에 파급 -> "가벼운 입력, 무거운 결과"의 아이러니

**Comment Corp에 적용 가능한 포인트:**
- 댓글 하나의 판단이 "더 큰 결과"로 이어지는 느낌 (예: 연쇄 이벤트, 셀럽 호감도 시스템)
- 숨겨진 정보의 재미: 현재 탐지기 아이템이 이 역할을 하지만, 더 강화 가능 (애매한 댓글 비중 증가)

### 1-2. Tinder 메카닉의 심리학

**왜 스와이프가 중독적인가:**
- **슬롯머신 효과**: 예측 불가능한 보상(매치)이 랜덤 간격으로 제공 -> 도파민 회로 자극
- **도파민은 보상이 아닌 기대에서 발생**: 신호(스와이프 가능한 카드)와 결과 사이의 "기대" 구간이 최대 도파민 구간
- **에이전시의 착각**: 한 손가락 움직임으로 운명을 결정하는 느낌 -> 과잉 통제감
- **무한 피드 패턴**: 끝이 없는 카드 -> "다음 한 장"에 대한 호기심

**Comment Corp에 적용 가능한 포인트:**
- 현재 120초 제한은 무한 피드와 반대 -> 이것이 오히려 "세션 종료 후 재도전" 욕구로 전환될 수 있음
- 스와이프 직후 보상 피드백(점수 팝업, 콤보 연출)의 즉시성이 핵심

## 2. 2024-2026 모바일 캐주얼 게임 트렌드

### 2-1. 하이브리드 캐주얼의 부상

**정의:** 하이퍼캐주얼의 단순한 코어 메카닉 + 미드코어의 진행/메타 시스템을 결합한 장르.

**핵심 수치 (2025):**
- 2027년까지 신규 출시 게임의 51%가 하이브리드 캐주얼로 예상
- 하이브리드 캐주얼 IAP 수익 H1 2025: $345M+ (전년 대비 대폭 증가)
- 성공 기준 리텐션: D7 15~20%, D30 10%
- Color Block Jam: 월 $14M 매출, 결정론적 "한 판만 더" 루프로 성공

**핵심 설계 원칙:**
1. 코어 루프는 30초 안에 이해할 수 있어야 함 (Comment Corp 충족)
2. 메타 레이어를 점진적으로 공개: 레벨 5까지는 코어만, 이후 진행 시스템 해금
3. 부스터와 보상형 광고의 적절한 배치
4. 시즌 이벤트 + 배틀패스로 장기 목표 제공
5. 옵션적 소셜(팀 경쟁, 리더보드)로 그룹 리텐션

### 2-2. 원탭 + 딥 메타 트렌드

2025년 가장 주목할 트렌드: "원탭 조작 + 깊은 메타 진행 시스템". Comment Corp의 스와이프 코어는 이 트렌드에 정확히 부합.

**메타 시스템 요소:**
- 언락 가능한 아이템/코스메틱 컬렉션
- 스트릭 보상 시스템
- 시즌 캘린더/일일 챌린지
- 메달/배지 수집

### 2-3. 수익화 트렌드

- 하이브리드: IAP 20~50% + 광고 수익 혼합
- "Skipits" 모델 (Mob Control): 광고 대신 구매 가능한 스킵 티켓
- 구독 모델: 광고 제거 + 일일 보상
- 배틀패스: 캐주얼 게임 66%가 채택, 가격은 저렴하게 ($1~5), 30~50배 가치 제공

## 3. 판단/분류 기반 게임 분석

### 3-1. Papers, Please (2013, Lucas Pope)

**핵심 메카닉:**
- 입국 심사관으로서 여권/서류 검증
- **속도 vs 정확성 트레이드오프**: 빨리 처리할수록 돈을 많이 벌지만 실수 확률 증가
- 점진적 규칙 복잡도 증가: 매일 새로운 규칙이 추가됨 (날인 추가, 국가 제한, 지문 대조 등)

**왜 재미있나:**
- **도덕적 딜레마**: 규칙을 따르면 무고한 사람이 피해 / 무시하면 자기 가족이 굶주림
- **절차적 긴장감**: 서류의 미세한 불일치를 찾는 "스팟 더 디퍼런스" 재미
- **피드백 명확성**: 실수 시 즉시 벌점, 3회 초과 시 급여 삭감

**Comment Corp에의 시사점:**
- 현재 게임은 "악플/선플 판단"이 핵심이지만, Papers Please처럼 점진적으로 판단 기준을 복잡하게 만들면 깊이 증가
- 예: 특정 키워드가 맥락에 따라 악플/선플이 되는 경우, 이모지로 위장한 악플, 이중 의미 댓글

### 3-2. Moderator Mayhem (2023, Engine/Techdirt)

**핵심 메카닉:**
- 콘텐츠 모더레이터 역할, 스와이프로 삭제/유지 결정
- **시간 압박 + 정확성 요구**: 빠르게 처리해야 승진, 하지만 정확도도 유지해야 함
- 레벨 진행 시 항소 심사, AI 모더레이션 조언 시스템 추가
- 맥락 조사 가능하지만 시간 소모

**Comment Corp과의 유사점 및 차이점:**

| 요소 | Moderator Mayhem | Comment Corp |
|------|-----------------|--------------|
| 코어 메카닉 | 스와이프 삭제/유지 | 스와이프 차단/승인 |
| 시간 압박 | 있음 (틱킹 클럭) | 있음 (120초 타이머) |
| HP 시스템 | 없음 (실적 기반) | 있음 (멘탈 게이지) |
| 진행 | 레벨별 규칙 추가 | 페이즈별 난이도 상승 |
| 서사 | 회사 풍자 | 셀럽 타입별 테마 |

### 3-3. Not Tonight (2018, PanicBarn)

**핵심 메카닉:**
- Papers Please 스타일의 신분증 검사 (클럽 바운서 역할)
- 나이, 만료일, 사진 대조, 금지 품목 등 다층 검증
- 라운드 사이에 아파트 관리, 이웃 상호작용, 스토리 분기

**교훈:**
- Papers Please와 동일 메카닉이지만 게임 길이가 너무 길어 지루해짐 (3~4시간 vs 90일)
- **Comment Corp의 120초 세션은 정확한 선택** -- 짧은 세션이 핵심
- 라운드 사이 메타 활동(스토리, 관리)이 리텐션에 도움

### 3-4. Y Social Media Moderator (itch.io)

- 소셜 미디어 모더레이터 시뮬레이터
- 유저 기분(mood) 게이지를 높게 유지하는 것이 목표
- Comment Corp의 멘탈 게이지와 유사한 개념

## 4. 리텐션 메카닉 심층 분석

### 4-1. 스트릭/일일 보상

**핵심 원리:**
- 손실 회피(loss aversion): 7일 이상 스트릭 유저는 일일 참여 확률 2.3배
- Duolingo: 7일 스트릭 유저의 코스 완료 확률 3.6배, 다음 날 복귀 확률 2.4배
- 스트릭+마일스톤 듀얼 시스템 사용 시 30일 이탈률 35% 감소

**베스트 프랙티스:**
- 스트릭 프리즈 (연속 실패 방지): Duolingo에서 이탈률 21% 감소
- 스트릭 램프다운: 끊어져도 0이 아닌 점진적 감소 -> 유저 좌절감 최소화
- Mistplay 사례: 주간 스트릭에서 일일 보상으로 전환 -> 유저 만족도 증가

### 4-2. 배틀패스/시즌패스

**캐주얼 게임 적용 원칙:**
- 코어 루프 강화형: 일반 플레이(스와이프 판단)가 곧 배틀패스 미션
- 짧은 시즌 (2~4주): 캐주얼 유저에게 적합
- 낮은 가격 ($1~3): 캐주얼 유저 지불 의향에 맞춤
- 30~50배 가치: 일반 구매 대비 압도적 가치 제공

### 4-3. 소셜 피처

**리텐션 영향:**
- 상위 모바일 게임 2/3이 최소 1개 소셜 피처 보유
- 커뮤니티 소속감 있는 유저: 게임 지속 확률 2.7배
- 친구 기반 리더보드가 글로벌 랭킹보다 리텐션 효과 높음
- 주기적 갱신되는 동적 리더보드가 침체 방지

## 5. 아이디어 도출: Comment Corp 메카닉 제안

> 각 아이디어에 다음 태그를 부여:
> - **구현 난이도**: 1(매우 쉬움) ~ 5(매우 어려움)
> - **재미 기대치**: 1(낮음) ~ 5(매우 높음)
> - **심플함 유지**: Y(코어 루프 변경 없음) / N(코어 루프 변경 필요)

---

### 5-A. 진행 시스템 (Progression)

#### A1. 셀럽 레벨업 & 언락 시스템
> 구현 난이도: 3 | 재미 기대치: 5 | 심플함 유지: Y

**설명:** 각 셀럽 타입(아이돌/배우/유튜버/스포츠/정치인)에 개별 레벨이 있음. 반복 플레이로 경험치 획득 -> 레벨업 시 해당 셀럽 전용 코스메틱(카드 테마, 배경, 효과음) 해금.

**설계 디테일:**
- 셀럽별 10레벨, 총 50레벨 진행
- 레벨 1: 기본 테마, 레벨 5: 전용 배경, 레벨 10: 전용 칭호 + 특수 댓글 풀 해금
- 누적 XP 기반 (플레이 점수의 10%가 XP로 변환)

**근거:** 하이브리드 캐주얼 트렌드의 핵심 -- 코어 루프 변경 없이 메타 레이어를 추가하여 장기 목표 제공. Reigns의 "죽어도 해금"과 같은 원리.

#### A2. 등급 시스템 (Rank Tier)
> 구현 난이도: 2 | 재미 기대치: 4 | 심플함 유지: Y

**설명:** 현재 S/A/B/C/D 등급을 누적 시스템으로 확장. 매 라운드 성적에 따라 RP(Rank Point) 획득 -> 브론즈/실버/골드/플래티넘/다이아 등급 진행.

**설계 디테일:**
- 각 등급 내 5단계 (브론즈 1~5, 실버 1~5...)
- 등급별 테두리/프로필 아이콘 변경
- 시즌(4주) 마다 소프트 리셋 (현 등급의 50% 유지)

**근거:** 짧은 세션 게임에서 장기 목표를 제공하는 가장 직관적 방법. 리더보드와 연동 시 효과 극대화.

#### A3. 셀럽 스토리 모드 (에피소드 해금)
> 구현 난이도: 4 | 재미 기대치: 5 | 심플함 유지: Y

**설명:** 각 셀럽 타입별 5~10개 에피소드. 특정 조건 달성 시(예: 아이돌 S등급 3회) 다음 에피소드 해금. 에피소드는 짧은 비주얼노벨 + 특수 조건 스테이지.

**설계 디테일:**
- 에피소드 도입: 30초 컷신 (셀럽의 상황 설명)
- 특수 조건: "이 라운드에서 악플을 1개도 놓치지 마세요" 등
- 에피소드 클리어 보상: 전용 아이템 스킨, 희귀 칭호

**근거:** Not Tonight의 라운드 간 스토리 + Papers Please의 도덕적 서사를 결합. 짧은 세션을 유지하면서도 "다음 이야기가 궁금해서 한 판 더" 유도.

### 5-B. 챌린지 & 미션 시스템

#### B1. 일일 챌린지 (Daily Challenge)
> 구현 난이도: 2 | 재미 기대치: 4 | 심플함 유지: Y

**설명:** 매일 고정 시드로 생성되는 특수 라운드. 모든 유저가 동일한 댓글 시퀀스를 받음 -> 공정한 일일 스코어 경쟁.

**설계 디테일:**
- 하루 1회 참여, 결과는 일일 리더보드에 반영
- 특수 조건 랜덤 부여: "오늘은 스킵 금지" / "콤보 30 달성 시 보너스 2배"
- 보상: 일일 코인 + 스트릭 보너스

**근거:** Slay the Spire의 데일리 클라임과 동일 원리. 공정한 경쟁 환경이 "자기 기록 깨기" 동기 강화.

#### B2. 주간 미션 세트
> 구현 난이도: 2 | 재미 기대치: 3 | 심플함 유지: Y

**설명:** 매주 3~5개 미션 세트 제공. 예: "유튜버 셀럽으로 3회 플레이", "콤보 50 달성", "멘탈 80 이상으로 생존"

**설계 디테일:**
- 미션 완료 시 별(Star) 획득
- 주간 별 누적으로 보상 단계 해금 (3별: 코인, 5별: 아이템, 7별: 코스메틱)
- 배틀패스와 연동 가능

**근거:** 배틀패스의 "코어 루프 강화형" 미션 설계. 유저가 평소 안 하던 셀럽 타입이나 플레이 스타일을 시도하게 유도.

#### B3. 엔들리스 모드 (Survival)
> 구현 난이도: 2 | 재미 기대치: 4 | 심플함 유지: Y

**설명:** 120초 제한 없이 멘탈이 0이 될 때까지 플레이. 시간이 지날수록 난이도 무한 상승.

**설계 디테일:**
- 30초마다 난이도 1단계씩 증가 (페이즈 무한 반복)
- 120초 이후부터는 기존 게임에 없는 고난이도 댓글 등장
- 생존 시간 기반 리더보드
- 아이템은 일정 시간마다 1개씩 랜덤 지급

**근거:** 120초 모드와 차별화된 "내가 얼마나 버틸 수 있는가"의 자기 도전. 타임어택과 서바이벌의 이중 모드는 리플레이 가치를 2배로 증가.

### 5-C. 코어 메카닉 심화

#### C1. 애매한 댓글 (Ambiguous Comments) 비중 확대
> 구현 난이도: 2 | 재미 기대치: 5 | 심플함 유지: Y

**설명:** "맥락에 따라 악플/선플이 달라지는" 댓글 카테고리 추가. 현재 binary(toxic/positive)에 "ambiguous" 타입을 추가.

**설계 디테일:**
- ambiguous 댓글: 좌/우 어느 쪽이든 50% 확률로 정답 처리 (혹은 맥락 힌트 기반 판단)
- 탐지기 아이템 사용 시 정답이 공개되므로 탐지기의 가치 급상승
- 고난이도 페이즈일수록 ambiguous 비율 증가 (Phase 4~5에서 20~30%)

**근거:** Papers Please의 핵심 재미 -- "규칙이 애매한 상황에서의 판단". 현재 게임의 단순 이진 분류에 깊이를 더함. Reigns의 "숨겨진 정보" 설계와 동일 원리.

#### C2. 연쇄 이벤트 (Chain Events)
> 구현 난이도: 3 | 재미 기대치: 4 | 심플함 유지: Y

**설명:** 특정 댓글 처리 결과가 다음 댓글에 영향을 주는 연쇄 시스템. 예: "팬 vs 안티 논쟁" 이벤트 -- 팬 댓글을 승인하면 안티 댓글이 연속으로 등장.

**설계 디테일:**
- 3~5개 댓글로 이루어진 미니 스토리라인
- 연쇄 이벤트 올클리어 시 보너스 점수 (콤보 x2 효과)
- 셀럽 타입별 고유 연쇄 이벤트 (아이돌: 팬덤 전쟁, 정치인: 여론 분열 등)

**근거:** Reigns의 멀티스텝 이벤트 체인. 단순 1회성 판단이 아닌 "이 선택이 다음에 무엇을 가져올까"의 전략적 판단 추가.

#### C3. 댓글 속성 확장 (다차원 판단)
> 구현 난이도: 4 | 재미 기대치: 4 | 심플함 유지: N

**설명:** toxic/positive 외에 "스팸", "광고", "허위정보", "비꼼(sarcasm)" 등 하위 분류 추가. 각 타입별 처리 방법이 다름.

**설계 디테일:**
- 차단(좌): 악플/스팸/허위정보에 적합
- 승인(우): 선플에 적합
- 위로 스와이프: 신고 (허위정보 전용, 보너스 점수)
- 아래로 스와이프: 무시 (비꼼 댓글, 차단/승인 모두 감점)

**근거:** Moderator Mayhem의 "컨텍스트 조사" 시스템을 간소화. 4방향 스와이프는 Reigns 후속작에서도 시도된 바 있음. 단, 코어 루프가 복잡해지므로 상위 난이도/특수 모드에서만 적용 권장.

#### C4. 좋아요/싫어요 수 기반 전략적 판단
> 구현 난이도: 1 | 재미 기대치: 3 | 심플함 유지: Y

**설명:** 현재 likes는 보이지 않는 내부 변수인데, 이를 카드에 표시. 좋아요 수가 높은 악플을 차단하면 더 높은 점수, 하지만 놓치면 더 큰 멘탈 데미지.

**설계 디테일:**
- 카드 하단에 좋아요/싫어요 수 표시 (좋아요 높을수록 카드 테두리 빨갛게)
- 좋아요 높은 악플 = "인기 악플" -> 차단 시 보너스 x2
- 좋아요 높은 선플 = "인기 선플" -> 승인 시 멘탈 회복 2배

**근거:** 이미 시스템에 likes_min/likes_max가 존재하므로 구현 비용 최소. 시각적 피드백 추가로 전략적 판단의 깊이 증가.

### 5-D. 메타 시스템 & 수집

#### D1. 아이템 상점 & 코인 이코노미
> 구현 난이도: 3 | 재미 기대치: 4 | 심플함 유지: Y

**설명:** 게임 플레이로 코인 획득 -> 상점에서 아이템(탐지기/프리즈/부스트/스킵 추가 수량), 코스메틱(카드 테마, 배경) 구매.

**설계 디테일:**
- 매 라운드 점수의 1/10이 코인으로 변환
- 일일 챌린지 완료, 미션 달성 등으로 보너스 코인
- 아이템 가격 밸런스: 탐지기(100코인) < 프리즈(200) < 부스트(150) < 스킵(250)
- 코스메틱은 높은 가격대 (500~2000코인)

**근거:** 하이브리드 캐주얼의 필수 요소. 코어 루프의 보상을 메타 레이어의 목표와 연결.

#### D2. 칭호 & 업적 시스템
> 구현 난이도: 2 | 재미 기대치: 3 | 심플함 유지: Y

**설명:** 특정 조건 달성 시 칭호 획득. 프로필에 장착 가능.

**설계 디테일 (예시 칭호):**
- "악플 사냥꾼": 악플 500개 차단
- "멘탈 갑": 멘탈 90 이상으로 120초 생존
- "콤보 마스터": 콤보 100 달성
- "만렙 매니저": 모든 셀럽 레벨 10 달성
- "피버 중독": 한 라운드에서 피버 3회 발동
- 희귀/전설 칭호: 숨겨진 조건 (유저 탐색 유도)

**근거:** 컬렉션 욕구 자극 + 장기 목표 제공. 구현 비용 대비 리텐션 효과가 매우 높은 메카닉.

#### D3. 카드 도감 (Comment Archive)
> 구현 난이도: 2 | 재미 기대치: 3 | 심플함 유지: Y

**설명:** 게임에서 만난 모든 댓글을 도감에 기록. 셀럽별/난이도별/태그별 정렬 가능.

**설계 디테일:**
- 댓글 첫 등장 시 "NEW!" 표시
- 도감 완성도 % 표시 (셀럽별)
- 도감 완성 마일스톤 보상: 25% -> 아이템, 50% -> 코스메틱, 100% -> 전설 칭호
- 댓글 카드 뒤집기: 해당 댓글의 실제 좋아요 수, 정답률 통계 확인 가능

**근거:** 포켓몬 도감 효과 -- "다 모아야 해"의 수집 본능. 기존 댓글 데이터를 활용하므로 추가 콘텐츠 비용 낮음.

### 5-E. 소셜 & 경쟁

#### E1. 친구 리더보드 & 주간 랭킹
> 구현 난이도: 3 | 재미 기대치: 4 | 심플함 유지: Y

**설명:** 친구 목록 기반 리더보드. 주간 단위 리셋.

**설계 디테일:**
- 로그인 연동 (Apple/Google)
- 친구 초대 보상 (양쪽 모두 코인 지급)
- 주간 상위 3명에게 특별 보상
- 글로벌 리더보드는 별도 탭 (등급별 필터)

**근거:** 친구 기반 리더보드가 글로벌 랭킹보다 리텐션에 효과적. 주간 리셋으로 항상 경쟁 가능한 상태 유지.

#### E2. 비동기 대전 (Ghost Race)
> 구현 난이도: 4 | 재미 기대치: 5 | 심플함 유지: Y

**설명:** 다른 유저의 기록된 플레이를 "고스트"로 나란히 보며 동시 플레이. 실시간은 아니지만 대전 느낌.

**설계 디테일:**
- 상대 고스트의 점수/콤보가 실시간으로 화면 상단에 표시
- 동일한 댓글 시퀀스를 받음 (고정 시드)
- 승리 시 보너스 코인, 연승 스트릭 보너스
- 매칭은 비슷한 등급끼리

**근거:** 실시간 멀티플레이어 없이도 경쟁 경험 제공. 구현 비용은 서버에 플레이 로그만 저장하면 됨.

#### E3. 길드/팀 시스템 (간소화)
> 구현 난이도: 5 | 재미 기대치: 3 | 심플함 유지: Y

**설명:** 3~5인 팀을 구성하여 주간 팀 점수 합산 경쟁.

**설계 디테일:**
- 팀 생성/가입만 가능 (채팅 없이 간소화)
- 팀 점수 = 멤버 주간 최고 점수 합산
- 팀 랭킹 상위권 보상: 팀 전용 배지
- 팀 미션: "팀원 전원이 이번 주 S등급 달성"

**근거:** 상위 모바일 게임 70%가 길드 피처 보유, 커뮤니티 소속감 유저의 지속 확률 2.7배. 단, Comment Corp의 캐주얼 특성상 최소화 버전이 적합.

**설명:** 매일 1회 플레이 시 출석 도장. 연속 출석 시 보상 증가.

**설계 디테일:**

**근거:** Duolingo 스트릭 모델 검증 완료. 7일 스트릭 유저의 다음 날 복귀 확률 2.4배. 스트릭 프리즈로 이탈률 21% 감소.

#### F2. 일일 무료 아이템

**설명:** 매일 첫 플레이 시 랜덤 아이템 1개 무료 지급.

**설계 디테일:**
- 로그인 시 아이템 가챠 연출 (룰렛/카드 뒤집기)
- 확률: 탐지기 40%, 프리즈 25%, 부스트 25%, 스킵 10%
- 광고 시청 시 1개 추가

**근거:** 가챠/룰렛의 시각적 재미 + 매일 로그인 유인. 아이템이 게임 내 즉시 효용 -> 받자마자 "한 판 하고 싶다" 유도.

### 5-G. 라이브 이벤트 & 시즌

#### G1. 시즌 테마 이벤트
> 구현 난이도: 3 | 재미 기대치: 4 | 심플함 유지: Y

**설명:** 2~4주 단위 시즌 테마. 예: "연말 시상식 시즌" -- 특수 댓글 풀 + 한정 보상.

**설계 디테일:**
- 시즌별 특수 댓글 20~30개 추가 (기존 풀에 혼합)
- 시즌 포인트: 일반 플레이와 별도로 시즌 전용 포인트 누적
- 시즌 보상 트랙: 무료 트랙 + 프리미엄 트랙 (배틀패스)
- 시즌 종료 후 해당 코스메틱은 획득 불가 (FOMO 유발)

**근거:** 캐주얼 게임 66%가 배틀패스 채택, 시즌제 운영이 장기 리텐션의 핵심.

#### G2. 실시간 이슈 연동 이벤트 (주의 필요)
> 구현 난이도: 5 | 재미 기대치: 5 | 심플함 유지: Y

**설명:** 실제 연예/스포츠 이슈에 맞춘 한정 이벤트. 예: 올림픽 기간 -> 스포츠 셀럽 특별 모드.

**설계 디테일:**
- 서버 사이드 이벤트 푸시 (Firebase Remote Config 활용)
- 실제 이슈 참조하되, 특정 인물 언급은 회피 (법적 리스크)
- 이벤트 기간 해당 셀럽 타입 보상 2배

**주의사항:** 실제 인물/사건과의 연관은 법적/윤리적 리스크. 추상화된 "스포츠 시즌" 수준으로만 운영 권장.

**근거:** 현실 이벤트 연동은 최고의 화제성, 하지만 리스크도 최고. Papers Please가 현실 정치를 게임화한 것처럼, 적절한 추상화가 핵심.

### 5-H. 게임플레이 변형 모드

#### H1. 보스 라운드 (Boss Comment)
> 구현 난이도: 3 | 재미 기대치: 5 | 심플함 유지: Y

**설명:** 일반 댓글 사이에 "보스 댓글"이 등장. 매우 교묘한 악플로, 여러 번의 판단이 필요.

**설계 디테일:**
- 보스 댓글은 3단계로 구성: 첫 번째는 선플처럼 시작 -> 두 번째에서 의미가 반전 -> 세 번째에서 진짜 의도 공개
- 각 단계마다 차단/승인 선택 (총 3번 스와이프)
- 3단계 모두 정확히 판단하면 보너스 점수 x5 + 멘탈 회복
- 1개라도 틀리면 일반 실수와 동일 처리
- 라운드당 1~2회 등장 (60초, 100초 부근)

**근거:** Reigns의 멀티스텝 이벤트 + Papers Please의 다층 검증 결합. "보스전"의 긴장감을 스와이프 메카닉에 녹여낸 독창적 변형.

#### H2. 스피드 모드 (Speed Run)
> 구현 난이도: 1 | 재미 기대치: 3 | 심플함 유지: Y

**설명:** 60초 타임어택. 모든 것이 2배속. 고정 댓글 수(50개)를 최대한 빨리 처리.

**설계 디테일:**
- 60초 제한 + 50개 댓글 고정
- 전부 처리하면 남은 시간이 보너스 점수로 변환
- 전용 리더보드

**근거:** 기존 120초 모드의 변형. 코어 로직 변경 최소화하면서 다른 플레이 경험 제공.

#### H3. 역전 모드 (Reversed)
> 구현 난이도: 2 | 재미 기대치: 4 | 심플함 유지: Y

**설명:** 악플을 승인하고 선플을 차단해야 하는 반전 모드. 설정: "악플러의 시점에서 인터넷을 어지럽혀라!"

**설계 디테일:**
- 좌(차단) = 선플 제거, 우(승인) = 악플 통과
- 멘탈 게이지 대신 "혼돈 게이지" (높을수록 좋음)
- 기존 직관과 반대 -> 인지 부조화로 독특한 재미

**근거:** 간단한 규칙 반전이지만 인지 부하가 크게 달라짐. 기존 유저에게도 신선한 챌린지.

#### H4. 하드코어 모드 (No Items, High Speed)
> 구현 난이도: 1 | 재미 기대치: 3 | 심플함 유지: Y

**설명:** 아이템 없이, Phase 3 속도로 시작하는 고난이도 모드.

**설계 디테일:**
- 아이템 0개, 멘탈 초기값 70, 악플 비율 60% 고정
- 생존 자체가 목표 (S등급 기준 상향)
- 전용 칭호 보상: "하드코어 서바이버"

**근거:** 코어 유저를 위한 도전 모드. balance.json 파라미터 조정만으로 구현 가능.

### 5-I. UX/피드백 강화

#### I1. 스와이프 판정 연출 강화
> 구현 난이도: 2 | 재미 기대치: 4 | 심플함 유지: Y

**설명:** 정답/오답 시 화면 전체에 걸친 피드백 강화.

**설계 디테일:**
- 정답 시: 카드가 시원하게 날아감 + 점수 팝업 + 콤보 카운터 애니메이션
- 오답 시: 화면 흔들림(shake) + 멘탈 게이지 깜빡임 + 카드 깨지는 효과
- 피버 진입 시: 배경 색상 변경 + 파티클 이펙트 + 진동(haptic)
- 콤보 10/20/50 달성 시: 화면 중앙 대형 텍스트 팝업 ("AMAZING!", "INCREDIBLE!")

**근거:** Tinder의 매치 연출처럼, 스와이프 결과의 감각적 피드백이 도파민 루프를 강화. 현재 플래시 오버레이만 있는 것을 대폭 강화.

#### I2. 댓글 카드 비주얼 다양화
> 구현 난이도: 2 | 재미 기대치: 3 | 심플함 유지: Y

**설명:** 댓글 카드의 시각적 다양성 증가.

**설계 디테일:**
- 프로필 아이콘 랜덤 생성 (익명 실루엣 + 색상 조합)
- 좋아요 수에 따른 카드 테두리 색상 변화
- 셀럽 타입별 카드 배경 테마
- 이벤트 댓글은 특수 디자인 (번쩍이는 테두리)
- 난이도별 미세한 시각 단서 (고난이도 = 더 리얼한 프로필)

**근거:** 시각적 다양성이 "같은 것의 반복" 느낌을 줄여 세션 내 지루함 방지.

## 7. 추천 구현 로드맵

### Phase 1: Quick Wins (1~2주)
> 코어 루프 변경 없이 기존 시스템 활용

1. **C4. 좋아요 수 시각화** - 이미 데이터 존재, UI만 추가
2. **C1. 애매한 댓글** - 댓글 데이터에 ambiguous 타입 추가 + 판정 로직 수정
3. **I1. 스와이프 연출 강화** - 기존 플래시 오버레이를 풍부한 애니메이션으로 교체
4. **H2/H4. 스피드/하드코어 모드** - balance.json 파라미터만 변경

### Phase 2: 메타 레이어 (2~4주)
> 코어 루프 밖의 진행/보상 시스템

5. **F1. 출석/스트릭 보상** - SharedPreferences + 간단한 UI
6. **D2. 칭호/업적** - 조건 체크 로직 + 프로필 UI
7. **A2. 등급 시스템** - RP 누적 + 등급 UI
8. **B1. 일일 챌린지** - 일일 시드 기반 댓글 큐 + 리더보드

### Phase 3: 콘텐츠 확장 (4~8주)
> 게임 깊이와 다양성 증가

9. **H1. 보스 라운드** - 멀티스텝 댓글 시스템 설계 + 구현
10. **C2. 연쇄 이벤트** - 이벤트 체인 데이터 구조 + 트리거 로직
11. **A1. 셀럽 레벨업** - XP 시스템 + 언락 관리
12. **B3. 엔들리스 모드** - 타이머 로직 변경 + 무한 난이도 스케일링
13. **H3. 역전 모드** - 판정 로직 반전 + 전용 UI 테마

### Phase 4: 소셜 & 라이브 (8~12주)
> 서버 의존 피처

14. **E1. 친구 리더보드** - Firebase Auth + Firestore 연동
15. **D1. 코인 이코노미** - 경제 밸런스 설계 + 상점 UI
16. **G1. 시즌 이벤트** - Remote Config + 시즌 콘텐츠 파이프라인
17. **E2. 비동기 대전** - 플레이 로그 저장 + 매칭 로직
