### 1위: 엔딩 카드 수집 시스템 (총점 8.5)

- **설명**: 120초 라운드 종료 후, 플레이 결과에 따라 고유한 "엔딩 카드"를 획득하는 시스템. 25종 이상의 엔딩 카드가 존재하며, 각각 특정 조건을 달성해야 해금된다. 도감 화면에서 수집 현황을 확인할 수 있고, 미발견 카드는 실루엣으로 표시되어 도전 욕구를 자극한다.
- **엔딩 카드 예시**:
  - "완벽한 매니저" -- 정확도 100% 달성
  - "콤보 장인" -- 최대 콤보 50 이상
  - "철벽 방어" -- 멘탈 90% 이상 유지하며 클리어
  - "기사회생" -- 멘탈 5% 이하에서 타임 클리어 생존
  - "속도의 왕" -- 80개 이상 댓글 처리
  - "무자비한 검열관" -- 히든 엔딩: 모든 댓글을 차단
  - "관대한 매니저" -- 히든 엔딩: 모든 댓글을 승인
  - "피버 마스터" -- 피버 3회 이상 발동
  - "무아이템 클리어" -- 아이템 0회 사용으로 S등급
  - "10초의 기적" -- 마지막 10초에 10000점 이상 획득
  - "멘탈 브레이크" -- 10초 내에 멘탈 0 (역설적 도전)
  - "정치인 스페셜리스트" -- 정치인 모드에서 S등급
  - 등 총 25종
- **구현 방법**:
  - `EndingCard` 모델 클래스: id, name, description, condition, iconAsset, isUnlocked
  - `EndingEvaluator` 서비스: 게임 종료 시 GameState를 분석하여 해당하는 모든 엔딩 조건을 체크
  - SharedPreferences 또는 Hive로 로컬 저장 (해금된 엔딩 카드 목록)
  - 도감 화면: GridView로 카드 목록 표시, 해금/미해금 상태 분리
  - 결과 화면에 "NEW!" 배지와 함께 새로 획득한 엔딩 카드 팝업
- **예상 효과**:
  - 리플레이 가치 극대화: "이 엔딩 카드를 모으려면 어떻게 플레이해야 하지?"
  - Papers Please의 다중 엔딩 수집 심리를 활용한 장기 리텐션
  - SNS 공유 시 "나 히든 엔딩 찾았다!" 같은 자발적 바이럴
  - 25종 완수 시 특별 보상(골든 카드 프레임 등)으로 컴플리셔니스트 충족
- **우선순위**: v1.1

### 2위: 데일리 챌린지 + 스트릭 통합 (총점 8.2)

- **설명**: 매일 날짜 기반 시드(seed)로 고정된 댓글 세트 + 특수 조건이 생성되어 전 세계 유저가 동일한 조건에서 경쟁하는 시스템. 연속 플레이 시 스트릭 보너스가 누적되며, 별도의 데일리 리더보드에서 순위를 확인한다.
- **데일리 조건 예시**:
  - "오늘은 유튜버 댓글만, 악플 비율 70%"
  - "속도 1.5배, 아이템 사용 불가"
  - "멘탈 시작 50%, 힐 2배"
  - "피버 없음, 대신 모든 점수 1.5배"
- **스트릭 보상 구조**:
  - 3일 연속: 랜덤 아이템 +1
  - 5일 연속: 인게임 코인 2배
  - 7일 연속: 주간 보스전 해금 + 특별 카드 프레임
  - 14일 연속: "데일리 마스터" 칭호
  - 30일 연속: 레전더리 엔딩 카드 "근면한 매니저" 해금
- **스트릭 깨짐 완화**:
  - 하루 놓치면 1일만 후퇴 (7일 -> 6일, 완전 리셋 아님)
  - 리워드 광고 시청으로 놓친 날 1회 복구 가능
  - "보험 티켓" (인게임 코인 구매)으로 하루를 사전 보호
- **구현 방법**:
  - 날짜 시드: `DateTime.now()`의 연-월-일을 시드로 Random 생성 -> 댓글 순서 및 특수 조건 결정
  - 데일리 리더보드: Firebase Firestore 컬렉션 (dailyScores/{YYYY-MM-DD}/scores)
  - 스트릭 추적: 로컬 SharedPreferences + 서버 동기화
  - 푸시 알림: "오늘의 데일리 챌린지가 기다립니다! 스트릭 8일째!" (Firebase Cloud Messaging)
- **예상 효과**:
  - D1 리텐션 35-40% 목표 달성의 핵심 동력
  - Wordle과 동일한 "오늘 다같이 같은 걸 한다"는 공유 문화 형성
  - 스트릭이 손실 회피 심리를 활용하여 매일 복귀 동기 부여
  - 데일리 리더보드가 경쟁심 자극
- **우선순위**: v1.2

### 3위: 스토리 이벤트 시스템 (총점 8.2)

- **설명**: 게임 도중 랜덤 타이밍에 "뉴스 속보" 형태의 이벤트가 발생하여 댓글의 양상이 급변하는 시스템. 매 판마다 5-7개 이벤트 풀에서 2-3개가 랜덤 발동되어 매번 다른 드라마를 경험하게 한다. 현재 EventService를 확장하는 형태로 구현.
- **이벤트 예시**:
  - "BREAKING: 셀럽 열애설 폭로!" -- 10초간 악플 비율 80%, 댓글 속도 1.5배
  - "셀럽 자선활동이 화제!" -- 10초간 선플 비율 90%, 포인트 2배
  - "안티팬 조직적 공격 감지!" -- 15초간 댓글 속도 2배, 난이도 +2
  - "셀럽이 직접 댓글창 등장!" -- 8초간 특수 댓글(셀럽 본인 응원 댓글) 등장, 승인하면 대량 보너스
  - "서버 과부하!" -- 5초간 댓글이 화면에 2장씩 동시에 등장 (빠른 판단 필요)
  - "커뮤니티 정화 운동!" -- 12초간 정답 시 멘탈 회복량 3배
  - "실시간 검색어 1위!" -- 8초간 좋아요 수가 10배 (점수에 큰 영향)
- **구현 방법**:
  - 기존 `EventService` 확장: `GameEvent` 모델에 `newsHeadline` 필드 추가
  - 이벤트 풀 JSON 파일에 20-30개 이벤트 정의
  - 발동 조건: 경과 시간 기반 확률적 트리거 (기존 시스템 활용)
  - UI: 화면 상단에 "BREAKING NEWS" 배너 슬라이드인 + 뉴스 텍스트 스크롤
  - 효과 적용: 기존 `speedMultiplier`, `toxicRatioOverride` 활용 + 새 필드 추가
- **예상 효과**:
  - 120초가 "미니 드라마"처럼 느껴짐 -- 매 판마다 다른 스토리 전개
  - Reigns의 "적응형 서사"와 유사한 효과를 아케이드 형태로 달성
  - 스와이프 피로 완전 방지 -- 이벤트가 리듬을 깨뜨리며 새로운 긴장 생성
  - "이번에 무슨 이벤트가 나올까?" 기대감으로 리플레이 유도
- **우선순위**: v1.1

### 4위: Wordle 스타일 결과 카드 (총점 8.1)

- **설명**: 게임 결과를 스포일러 없이 이모지 그리드로 변환하여 SNS에 공유하는 시스템. 120초를 12구간(각 10초)으로 나누어 각 구간의 퍼포먼스를 색상 블록으로 시각화한다.
- **결과 카드 포맷**:
  ```
  댓글주식회사 Day 142 | 아이돌
  S등급 | 42,350점
  🟩🟩🟩🔥🟩🟩🟥🟩🟩🔥🟩🟩
  콤보 45 | 정확도 92% | 멘탈 78%
  [play.commentcorp.app/daily/142]
  ```
  - 🟩 = 해당 구간 정답 80%+
  - 🟨 = 해당 구간 정답 50-79%
  - 🟥 = 해당 구간 정답 50% 미만
  - 🔥 = 해당 구간 피버 활성화
  - 💀 = 멘탈 10% 이하 구간
- **구현 방법**:
  - `GameNotifier`에서 10초 단위 구간별 정답/오답 카운트 누적
  - `ShareService`: 구간별 데이터를 이모지 문자열로 변환
  - Flutter `share_plus` 패키지로 텍스트 공유 (클립보드 복사 + 시스템 공유 시트)
  - 결과 화면에 "결과 공유" 버튼 추가 (현재 "다시 플레이" 위에 배치)
  - 옵션: 이미지 카드 생성 (RepaintBoundary -> 이미지 저장 -> 공유)
- **예상 효과**:
  - Wordle이 증명한 "공유 가능한 결과"의 바이럴 파워
  - "나는 🔥🔥이 2번 나왔는데 너는?" 같은 SNS 대화 유발
  - 유저 획득 비용(CPI) 절감에 직접 기여
  - 데일리 챌린지와 결합하면 "같은 날 같은 문제"의 비교 효과 극대화
- **우선순위**: MVP (즉시 구현)

### 5위: 주간 보스전 (총점 8.0)

- **설명**: 7일 데일리 챌린지 스트릭 달성 시 해금되는 특별 모드. "대형 안티팬 계정"이 보스로 등장하며, 일반 모드와 다른 긴장감 있는 서바이벌 형식으로 진행된다. 주 1회 도전 가능.
- **보스전 구조**:
  - 시간: 90초 (일반보다 짧지만 더 빡빡)
  - 페이즈 1 (0-30초): 보스가 "정찰" -- 일반 속도지만 난이도 높은 댓글만
  - 페이즈 2 (30-60초): 보스 "본격 공격" -- 속도 2배, 악플 비율 80%
  - 페이즈 3 (60-90초): 보스 "최종 폭주" -- 속도 3배, 모든 댓글 최고 난이도
  - 보스 HP 바: 정답 판정마다 보스 HP 감소. 90초 내에 보스 HP를 0으로 만들면 클리어
  - 보스마다 고유 패턴: "외모 악플 전문 안티", "루머 유포 전문 안티", "팬덤 내부 갈등 유발자"
- **보상**:
  - 클리어: 대량 코인 + 보스 전용 엔딩 카드 + 특별 칭호 ("안티 헌터")
  - 주간 보스 전원 클리어 글로벌 카운트 표시
- **구현 방법**:
  - `BossGameState` 확장: 보스 HP, 보스 페이즈, 보스 타입 추가
  - 보스별 댓글 풀: 특화된 고난도 댓글 세트 JSON
  - 보스전 전용 UI: 상단에 보스 아바타 + HP 바, 화면 양쪽에 공격 이펙트
  - 진입 조건: 스트릭 7일 체크 -> 보스전 버튼 활성화
- **예상 효과**:
  - "이번 주 보스가 뭐지?" 궁금증으로 매주 복귀 유도
  - 스트릭 시스템의 "킬러 보상" -- 7일 연속 플레이의 이유
  - 보스 클리어 영상이 TikTok/YouTube 콘텐츠로 활용 가능
  - 기존 아케이드 게임플레이에 RPG적 "보스 토벌" 긴장감 추가
- **우선순위**: v2.0

### 6위: 크리티컬 정답 시스템 (총점 7.9)

- **설명**: 정답 판정 시 일정 확률로 "CRITICAL!" 이 발동되어 해당 댓글의 점수가 대폭 증가하는 시스템. 가변 보상 스케줄(Variable Reward Schedule)을 적용하여 매 스와이프에 기대감을 부여한다.
- **상세 설계**:
  - 기본 크리티컬 확률: 15%
  - 크리티컬 배수: 점수 x5
  - 콤보 20 이상: 크리티컬 확률 20%로 상승
  - 피버 모드: 크리티컬 확률 30%
  - "SUPER CRITICAL" (확률 3%): 점수 x10 + 멘탈 전체 회복
  - 연출: 금색 이펙트 + "CRITICAL!" 텍스트 팝업 + 특수 효과음 + 짧은 화면 흔들림
- **구현 방법**:
  - `_handleCorrect()` 메서드에 확률 체크 로직 추가
  - `GameState`에 `criticalCount` 필드 추가
  - 크리티컬 발동 시 점수 계산에 크리티컬 배수 적용
  - UI: `AnimatedContainer` + `ShakeWidget`으로 크리티컬 연출
  - 결과 화면에 "크리티컬 X회" 통계 추가
- **예상 효과**:
  - 슬롯머신의 도파민 효과 -- "이번에 터질까?" 기대감
  - 하이라이트 모먼트 생성 -- 크리티컬 연속 발생 시 폭발적 점수 상승
  - 동일한 스킬 수준에서도 점수 변동이 생겨 "운 요소"로 인한 리플레이
  - 스트리밍 시 "크리티컬 떴다!" 같은 반응 유발
- **우선순위**: v1.1

### 7위: 라운드별 랜덤 규칙 모디파이어 (총점 7.7)

- **설명**: Papers Please에서 영감을 받아, 매 라운드 시작 전에 1-2개의 "특수 규칙"이 랜덤으로 부여되는 시스템. 규칙이 게임의 판단 기준을 변경하여 매판 다른 전략이 필요하게 한다.
- **규칙 모디파이어 예시** (풀에서 랜덤 1-2개 선택):
  - "좋아요 100 이상 댓글은 무조건 승인" (좋아요 수가 판단 기준에 추가)
  - "오늘은 반말 댓글만 차단" (존댓말 악플은 승인해야 함)
  - "연속 3번 같은 방향 스와이프 금지" (3번 연속 차단 시 페널티)
  - "처음 30초는 모든 댓글이 뒤집혀 있음" (텍스트를 볼 수 없고 프로필 정보만으로 판단)
  - "콤보 5마다 타이머 +2초" (콤보가 시간을 벌어줌)
  - "오답 시 즉시 다음 댓글 2배속" (실수의 대가가 속도 증가)
- **구현 방법**:
  - `RuleModifier` 모델: id, name, description, applyEffect(), 시각적 아이콘
  - 게임 시작 시 풀에서 1-2개 랜덤 선택 -> `GameState`에 `activeModifiers` 리스트 추가
  - 각 모디파이어의 효과를 `GameNotifier`의 해당 로직에 조건부 적용
  - UI: 게임 시작 전 "오늘의 규칙" 발표 화면 (1.5초 표시)
  - 게임 중 화면 상단에 활성 규칙 아이콘 표시
- **예상 효과**:
  - "이번에 무슨 규칙이 나올까?"로 매판 신선함 유지
  - Papers Please의 "점진적 복잡화" 긴장감을 캐주얼에 맞게 축소 적용
  - 규칙 조합이 만드는 시너지/갈등이 전략적 깊이 제공
  - 데일리 챌린지에 특정 규칙 조합을 고정하면 추가 변화
- **우선순위**: v2.0

### 8위: 비판 vs 악플 "그레이존" 강화 (총점 7.7)

- **설명**: 난이도 4-5에서 "정당한 비판"과 "악성 댓글"의 경계가 매우 모호한 "그레이존 댓글"을 추가. 이런 댓글은 이진적 정답/오답이 아닌 부분 점수 시스템을 적용하며, 판정 후 "커뮤니티 투표 결과"를 보여준다.
- **상세 설계**:
  - Comment 모델에 `grayZone: bool` 및 `communityToxicPercent: double` 필드 추가
  - 그레이존 댓글 예시: "솔직히 이 가수 노래 실력은 데뷔 수준도 안 됨. 연습을 더 했어야지" (비판 62% / 악플 38%)
  - 점수 계산: `communityToxicPercent`에 따라 부분 점수. 60% 악플인 댓글을 차단하면 60% 점수, 승인하면 40% 점수 (완전 오답이 아님)
  - 판정 후 0.5초간 "커뮤니티 투표: 62%가 악플로 판단" 오버레이 표시
  - 콤보: 그레이존 댓글은 다수 의견과 같은 방향이면 콤보 유지
- **구현 방법**:
  - 댓글 JSON에 그레이존 댓글 30-50개 추가 (난이도 4-5에 배치)
  - `_handleCorrect` / `_handleWrong` 분기에 그레이존 분기 추가
  - `communityToxicPercent`는 실제로는 사전 설정값 (서버 불필요)
  - UI: 판정 후 작은 원형 그래프 + 퍼센트 텍스트 오버레이 (0.5초 표시)
- **예상 효과**:
  - "내 판단은 맞았을까?" 성찰적 재미 -- 다른 게임에 없는 경험
  - 완전 오답의 좌절감 감소 (그레이존은 부분 점수이므로)
  - 사회적 메시지: "악플과 비판의 경계는 어디인가?"
  - 토론 유발: 결과 공유 시 "이 댓글을 너는 어떻게 판단했어?" 대화
- **우선순위**: v1.3

### 9위: 콤보 보험 (자동 1회) (총점 7.7)

- **설명**: 10콤보 이상 유지 시, 첫 번째 오답에서 콤보가 완전 리셋(0)되지 않고 현재의 절반으로 감소하는 자동 보험 시스템. 한 판에 1회만 발동되며, 소진 후에는 더 큰 긴장감을 제공한다.
- **상세 설계**:
  - 발동 조건: 콤보 >= 10 + 오답 발생 + 보험 미소진
  - 효과: combo = combo / 2 (반올림)
  - 연출: "COMBO SAVED!" 텍스트 + 방패 이펙트 + 노란색 플래시
  - 소진 후: HUD에 방패 아이콘이 깨진 상태로 표시 ("보험 없음" 인지)
  - 보험 2회차(유료): 광고 시청 또는 인게임 코인으로 "추가 보험 1회" 구매 가능
- **구현 방법**:
  - `GameState`에 `comboInsuranceUsed: bool` 필드 추가
  - `_handleWrong()`에서 조건 체크 -> 콤보 절반 처리
  - UI: HUD에 작은 방패 아이콘 (활성/비활성 상태)
- **예상 효과**:
  - "보험이 있으니까 좀 더 공격적으로 플레이" -> 보험 소진 -> "이제 진짜 조심해야 해" 감정 곡선 생성
  - 콤보 완전 리셋의 좌절감 1회 완화 -> 이탈 방지
  - 손실 회피 심리의 건강한 활용
- **우선순위**: v1.1

### 10위: 친구 도전 딥링크 (총점 7.7)

- **설명**: 결과 공유 시 딥링크를 포함하여, 링크를 받은 친구가 정확히 같은 댓글 세트로 플레이할 수 있게 하는 시스템. 양쪽 결과를 나란히 비교하는 "대결 결과" 화면을 제공한다.
- **상세 설계**:
  - 공유 시 시드(seed) 값을 URL 파라미터로 인코딩: `commentcorp.app/challenge?seed=ABC123&score=42350`
  - 링크 수신자가 앱에서 같은 시드로 플레이 -> 완료 후 대결 결과 화면
  - 대결 결과: 나란히 2열 비교 (점수, 콤보, 정확도, 등급)
  - "도전 수락" 시 별도 리더보드가 아닌 1:1 비교 화면
  - 앱 미설치자는 앱스토어로 리다이렉트 (유저 획득 효과)
- **구현 방법**:
  - Firebase Dynamic Links 또는 uni_links 패키지
  - 시드 기반 댓글 순서 결정 (기존 셔플 로직에 시드 파라미터 추가)
  - 대결 결과 화면: 2열 레이아웃 위젯
- **예상 효과**:
  - 비동기 멀티플레이어의 가장 간단한 형태
  - 직접적인 유저 획득 채널 (링크 클릭 -> 앱 설치)
  - "이 점수 넘어봐!" 같은 도전 심리 자극
- **우선순위**: v2.5

### 11위: 매니저 레벨 시스템 (총점 7.6)

- **설명**: 누적 플레이 점수로 "매니저 레벨"(1~30)이 상승하며, 레벨업 시 새로운 콘텐츠가 해금되는 장기 진행 시스템.
- **해금 구조**:
  - Lv.1: 아이돌 모드 (기본)
  - Lv.3: 배우 모드 해금
  - Lv.5: 새 아이템 "자동 필터" 해금
  - Lv.7: 유튜버 모드 해금
  - Lv.10: 하드 모드 (시간 90초, 속도 1.5배) 해금
  - Lv.13: 스포츠 모드 해금
  - Lv.15: 엔드리스 모드 (시간 무제한, 멘탈만으로 생존) 해금
  - Lv.18: 정치인 모드 해금
  - Lv.20: 라운드별 규칙 모디파이어 해금
  - Lv.25: 보스전 모드 해금
  - Lv.30: "사장님" 칭호 + 골든 프레임
- **구현 방법**:
  - `ManagerLevel` 모델: 레벨별 필요 누적 점수, 해금 콘텐츠 정의
  - SharedPreferences/Hive: 누적 점수 + 현재 레벨 저장
  - 레벨업 시 축하 화면 + 해금 콘텐츠 프리뷰
  - 메뉴 화면에 레벨 표시 + 진행률 바
- **예상 효과**:
  - 명확한 장기 목표 제공 -> "다음 레벨까지 얼마나 남았지?"
  - 콘텐츠 해금이 점진적이라 초보자 압도 방지 (온보딩 최적화)
  - 레벨이 높을수록 접근 가능한 콘텐츠가 많아져 이탈 비용 증가
- **우선순위**: v1.3

### 12위: 데일리 미션 3개 (총점 7.6)

- **설명**: 매일 3개의 작은 목표가 부여되고, 완료 시 인게임 코인 보상, 3개 전부 완료 시 보너스 상자를 획득하는 시스템.
- **미션 풀 예시** (매일 3개 랜덤 선택):
  - "아이돌 모드에서 30콤보 달성"
  - "정확도 85% 이상으로 클리어"
  - "프리즈 아이템 1회 이상 사용"
  - "B등급 이상 획득"
  - "60초 이상 생존"
  - "피버 1회 이상 발동"
  - "유튜버 모드에서 플레이"
  - "아이템 미사용으로 클리어"
  - "20개 이상 댓글 처리"
  - "멘탈 50% 이상 유지하며 클리어"
- **구현 방법**:
  - 미션 풀 20-30개 정의 -> 날짜 시드로 3개 선택
  - `DailyMission` 모델: id, description, checkCondition(GameState), isCompleted
  - 게임 종료 시 3개 미션의 조건을 GameState와 대조
  - 보상 지급: 인게임 코인 + 보너스 상자(랜덤 아이템/코스메틱)
- **예상 효과**:
  - 데일리 챌린지와 별개의 "매일 올 이유" 제공
  - "오늘 미션이 뭐지?" 궁금증으로 앱 실행 유도
  - 평소 안 하던 모드/전략을 시도하게 만들어 게임 경험 다양화
- **우선순위**: v1.2

### 13위: 멘탈 위기 연출 (총점 7.6)

- **설명**: 멘탈 게이지가 낮아질수록 시각/청각 연출이 변화하여 위기감을 극대화하는 시스템. 멘탈이 낮아도 생존하면 "기사회생" 쾌감을 제공한다.
- **연출 단계**:
  - 멘탈 50% 이하: 화면 가장자리에 미세한 붉은 비네팅
  - 멘탈 30% 이하: 비네팅 강화 + 배경음 BPM 1.3배 + 댓글 카드 미세 떨림
  - 멘탈 15% 이하: 강한 붉은 비네팅 + BGM 변조(저음 강화) + 심장박동 효과음 추가
  - 멘탈 5% 이하: 화면 전체 흑백 필터 + 심장박동 강렬 + "위험!" 텍스트 깜빡임
  - 멘탈 회복 시: 비네팅 서서히 사라짐 + "회복" 사운드
- **구현 방법**:
  - `GameScreen` 위젯에 멘탈 기반 오버레이 레이어 추가
  - `AnimatedContainer`로 비네팅 강도 조절
  - BGM: 2가지 트랙(평시/위기)을 크로스페이드 또는 피치/속도 조절
  - 심장박동: 멘탈 비례 간격으로 재생되는 효과음
- **예상 효과**:
  - Content Moderator Sim에서 입증된 "줄어드는 게이지"의 긴박감을 시각/청각으로 극대화
  - "멘탈 3%에서 살아남았다!"는 극적 경험 -> 공유 욕구 자극
  - 위기에서 회복하는 쾌감이 게임의 감정 곡선을 풍부하게 만듦
- **우선순위**: MVP (즉시 구현)

### 14위: 페이즈 전환 연출 (총점 7.5)

- **설명**: 현재 존재하는 5개 페이즈(0-30-60-90-110-120초) 전환 시점에 짧은 시각적 연출을 추가하여, 120초가 5개의 "챕터"처럼 느껴지게 하는 시스템.
- **연출 설계**:
  - 전환 시 0.3초 슬로모션(게임 속도 0.5배) + 화면 줌인 효과
  - "PHASE 2 -- 속도 UP!" 텍스트가 화면 중앙에 페이드인/아웃
  - 배경색이 페이즈별로 미묘하게 변화 (Phase 1: 차분한 블루 -> Phase 5: 강렬한 레드)
  - 각 페이즈에 이름 부여: "워밍업" -> "본격 시작" -> "가속" -> "폭풍" -> "최종 심판"
- **구현 방법**:
  - `GameNotifier`의 타이머 로직에서 페이즈 변경 감지 시 `phaseChanged` 플래그 설정
  - `GameScreen`에서 플래그 감지 시 오버레이 애니메이션 재생
  - 배경색: `AnimatedContainer`로 `BalanceConfig.phases`에 따른 색상 전환
  - 텍스트: `AnimatedOpacity` + `ScaleTransition`
- **예상 효과**:
  - 120초 내 "리듬"이 생겨 스와이프 피로 방지
  - "Phase 5까지 살아남는다"는 목표감 부여
  - 스트리밍 시 "와 Phase 5 진입!" 같은 하이라이트 모먼트 생성
- **우선순위**: MVP (즉시 구현)

### 15위: Near-miss 결과 화면 (총점 7.4)

- **설명**: 결과 화면에서 "조금만 더 했으면..." 느낌을 주는 Near-miss 정보를 표시하여 "한 판 더" 심리를 유도하는 시스템.
- **표시 예시**:
  - "A등급까지 3,200점 부족!" (다음 등급까지의 차이)
  - "최고 기록까지 1,500점!" (개인 최고 기록 대비)
  - "콤보 +3이면 새로운 최고 콤보!" (최대 콤보 Near-miss)
  - "정확도 2% 더 높았으면 S등급!" (정확도 기반 등급 Near-miss)
  - "15초만 더 버텼으면 타임 클리어!" (멘탈 사망 시)
- **구현 방법**:
  - `ResultScreen`에서 `GameState` 분석하여 가장 "아까운" Near-miss 1-2개 선택
  - 등급 기준값과 현재 점수의 차이 계산
  - 개인 최고 기록 (SharedPreferences) 대비 차이 계산
  - 텍스트를 결과 카드 하단에 강조색으로 표시
- **예상 효과**:
  - "거의 다 했는데" 심리로 즉시 재도전 유도
  - 구현이 매우 간단하면서 효과가 큼 (ROI 최고)
  - 첫 세션 -> 두 번째 세션 전환율 직접 향상
- **우선순위**: MVP (즉시 구현)

## Key Design Principles (핵심 디자인 원칙)

1. **스와이프는 성스럽다**: 좌/우 스와이프 코어 메카닉은 절대 변경하지 않음. 모든 신규 기능은 이 위에 쌓는다.
2. **120초의 밀도**: 한 판이 짧되, 그 안에서 드라마가 일어나야 한다 (이벤트, 페이즈, 크리티컬).
3. **한 판 더의 이유**: Near-miss, 엔딩 카드 미수집, 데일리 미션 미완료 등 항상 "다음 판"의 이유가 있어야 한다.
4. **심플한 겉, 깊은 속**: 인터페이스는 스와이프 하나지만, 내부에 콤보/크리티컬/보험/이벤트/규칙이 층층이 쌓여 깊이를 만든다.
5. **공유하고 싶은 결과**: Wordle 카드, 엔딩 카드, 크리티컬 연속 등 SNS에 올리고 싶은 순간을 만든다.
